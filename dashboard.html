          Fire drillPractice detecting and recovering from a real WAN outage. Run this when a brief internet interruption will not disrupt anyone.

          New here? Run the one-minute fire drill to watch Linkmoth catch and diagnose an outage live.

          Plan for a short interruption. This takes about a minute and briefly interrupts internet for everyone on this network.

          1. Unplug the router's WAN cable, the one coming from the modem or wall socket (not a LAN cable). Then press the first button.
          2. When Linkmoth has caught it, plug the cable back in and press the second button to verify recovery.

          Installation and updatesThe update check only runs when you press the button. GitHub receives this host's public IP, request time, ordinary TLS/HTTP metadata, and Linkmoth's generic User-Agent. Linkmoth sends no identifier, configuration, incident data, diagnostics, credentials, or usage telemetry.

          Evidence exportsExports are created locally from bounded evidence. Detailed JSON is for administrators; readable local is a plain-text handoff; support-safe JSON removes credentials and pseudonymizes private-network identifiers consistently within that export.
          
          
          
            
            
            
          
          
        
          
        
          
          
          BackupA portable snapshot of history, settings, and device/webhook configuration for moving to new hardware. Deliberately excludes secrets (TOTP seed, this installation's inbound webhook token, push key) and the TLS CA – and the included database snapshot is itself sanitized, clearing every outbound webhook's destination URL and custom headers (the webhooks are also left disabled), push subscriptions, and session/login state. Restoring on a new device means redoing the one-time setup (new password, TOTP, webhook secret) and re-entering then re-enabling each webhook's destination, via the CLI's --restore flag. Restoring requires the CLI (linkmoth.py --restore <file>) with the service stopped.
