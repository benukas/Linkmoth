      Browser push

      Loading push status…

      Works on desktop and Android in the browser. On iPhone/iPad you must add Linkmoth to the Home Screen first (Share → Add to Home Screen), open it from the icon, then enable push — Safari tabs cannot subscribe. The Linkmoth CA must be fully trusted on iOS (profile install and Settings → General → About → Certificate Trust Settings). Global outages defer push until the network recovers.
